# SVM_feature_combine
图片的多个特征合并为一个特征输入svm训练和测试<br>
combine two or more feature to train svm and test

### 环境env <br>
python2.7<br>
opencv-python<br>
numpy<br>

### 使用说明 help:<br>
使用
```Bash
python svm.py
```
查看使用说明并使用.<br>
try 
```Bash
python svm.py
```
to get help
